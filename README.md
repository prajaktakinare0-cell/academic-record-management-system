
# Academic Record Management System

## Project Description 
The Academic Record Management System is a Linux Bash Shell project used to manage student academic records through the terminal

## Features
- Add Student Record
- View All Records
- Search Record by Roll Number
- Update Student Record
- Delete Student Record
- Exit System

## Technologies Used
- Linux 
- Linux Bash shell scripting project

## Commands Used
- read
- echo
- case
- cat 
- grep
- sed

## How to Run
1) Give execution permission:
     chmod +x academic_record.sh
2) Run the project:
     ./academic_record.sh
